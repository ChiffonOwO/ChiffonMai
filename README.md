ChiffonMai 是一款专为 **舞萌DX 2025(maimai DX 2025)** 玩家打造的一站式移动端工具类应用，聚合了曲库查询、成绩统计、Rating计算、曲目推荐等多种实用功能，助力玩家提升游玩体验。

## ✨ 核心功能

| 功能模块 | 功能描述 |
|---------|---------|
| 🎵 乐曲查询 | 完整查询舞萌DX曲库的所有乐曲信息 |
| 📊 成绩查询 | 查看个人游玩数据、成绩记录 |
| 🏆 Best50查询 | 计算并展示个人Best50成绩 |
| 📈 拟合Best50查询 | 分析拟合Best50拟合情况 |
| 🎯 基于Best50推荐 | 根据你的Best50成绩智能推荐适合的曲目 |
| 🎼 基于流派推荐 | 按你常玩的曲目流派个性化推荐 |
| 🏷️ 基于标签推荐 | 基于谱面标签（如难度、风格）精准推荐曲目 |
| 🔀 随机乐曲 | 随机生成1-4首曲目 |
| 🧮 单曲Rating计算 | 计算单曲Rating |
| 📉 达成率计算 | 根据判定详情（Perfect/Great等）精准算出达成率 |
| 🆚 版本对照 | 版本名对照版本号，如maimai GreeN对应超代 |
| 🔍 达成率反推 | 根据达成率反推绝赞判定详情 |
| 📱 绑定二维码 | 关联你的舞萌DX账号，同步游戏数据 |
